## Overview
This project analyzes and visualizes a user dataset, focusing on demographics, BMI, age groups, and other features. 
Modern, visually appealing plots are included to make insights easy to understand.

## Features
- Clean and preprocess complex JSON data
- Calculate BMI and categorize users
- Group users by age ranges
- Visualizations include:
  - Scatter plots (Age vs Height/Weight)
  - Histograms (Age, Height, Weight)
  - Count plots (Gender, BMI Category, Blood Group, Eye Color)
  - Pie chart (Role distribution)
- Modern color palettes for professional visuals
